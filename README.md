Mad Libs Generator

A Mad Libs game that allows users to create funny stories by providing various parts of speech.

Features
Story Selection: Choose from 3 distinct themes or let the computer pick randomly.
Robust Input Handling: Includes a validation system that ensures:
  - Users cannot leave a field blank.
  - Fields requiring numbers actually receive digits.
Dynamic Formatting: Uses Python f-strings for clean story generation.
How to Run
1. Run the script using Python: `python mad_libs.py`
2. Enter a number (1-4) to select your story.
3. Follow the prompts to fill in the blanks.
4. Read your generated story!
